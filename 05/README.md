# グループプログラミング演習

役割分担で取り組むグループワーク（ライター／プログラマー／レビュアー）

## 概要

本演習では、グループでプログラミングに取り組みます。以下の3つの役割に分担して進めてください。

| 役割 | 担当内容 |
|------|----------|
| ライター | コードを書く担当 |
| プログラマー | 実装方針を考える担当 |
| レビュアー | コードを確認する担当 |

## 役割分担のルール

- それぞれの役割を遵守し、役割以上の内容は行わないでください。
- ライターはプログラムの内容や実装に関して話してはいけません。
- プログラムの内容は、あくまでプログラマーとレビュアーの話し合いによって決定してください。
- 役割をローテーションできるといいです。

### 課題2（余裕があるグループ向け）：2次方程式の解（`ex02_quadratic.py`）

2次方程式 `ax^2 + bx + c = 0` の解を求める関数 `calc_quadratic_equation(a, b, c)` を設計し、テストを行ってください。

- 判別式 `D` を用いて実装してください。虚数解は `"解なし"` と出力してください。
- 平方根計算には `math.sqrt(変数名)` が使えます。
- 解が2つの場合、`return (変数1, 変数2)` のように返してください（変数名はなんでもOK）。

```python
def calc_quadratic_equation(a, b, c):
    pass

assert calc_quadratic_equation(1, 0, -4) == (2.0, -2.0)   # 解が2つ
assert calc_quadratic_equation(1, -2, 1) == 1.0            # 重解
assert calc_quadratic_equation(1, 0, 4) == "解なし"        # 虚数解
```

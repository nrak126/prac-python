## 復習演習：九九表の作成 → `ex00_kuku.py`

### 問題
二重の `for` 文を使って、1の段から9の段までを表示する「九九表」を作成してください。

### 出力例
```
1	2	3	4	5	6	7	8	9	
2	4	6	8	10	12	14	16	18	
...
9	18	27	36	45	54	63	72	81	
```

### ヒント
- 外側の `for` 文で「段（i）」を、内側の `for` 文で「かける数（j）」を回す
- `print(値, end="\t")` でタブ区切りの出力ができる
- 内側のループが終わったあとに `print()` を呼ぶと改行できる

## 演習1：サイコロ → `ex01_dice.py`

### 問題
1から6の目が出るサイコロを10回振り、その全ての出目を1つのリストに格納して、最後にリストを表示するプログラムを作成してください。

### 出力例
```
[3, 1, 6, 2, 5, 4, 1, 3, 6, 2]
```

### ヒント
- `random.randint(1, 6)` で1〜6のランダムな整数を生成できる
- リストへの追加は `results.append(値)` を使う

### 問題

## 演習：モジュール分割 → `ex05_module/`

### 問題
`ex03_greeting.py` と `ex04_tax.py` で作成した関数（`say_hello`・`calc_tax`）を `utils.py` に実装し、`main.py` から `import` して呼び出してください。

### ファイル構成
```
ex05_module/
├── utils.py   ← 関数を定義する
└── main.py    ← utils から import して呼び出す
```

### ヒント
- `main.py` の先頭に `from utils import say_hello, calc_tax` と書く
- `main.py` は `ex05_module/` ディレクトリ内で実行する

## 演習：ネストされたデータの取得 → `ex07_nested_dict.py`

### 問題
以下のデータから、装備（`equipment`）の中にある「武器（`weapon`）」を表示してください。

```python
user_status = {
    "username": "suzuki_77",
    "equipment": {"weapon": "鉄の剣", "armor": "革の服"}
}
```

### 出力例
```
現在の武器: 鉄の剣
```

### ヒント
- ネストされた辞書は二段階でキーを指定する: `user_status["equipment"]["weapon"]`

## 演習：テスト結果の集計と探索 → `ex08_json/`

### 問題
`exam_results.json` を読み込み、以下の3つの処理を行ってください。

1. 受験者全員の **平均点** を計算して表示する
2. **最高得点** を取った人の「名前」と「点数」を表示する
3. 1・2の集計処理部分を、それぞれ **別の関数** として定義して呼び出す

### 使用するJSONデータ（`exam_results.json`）
```json
[
    {"name": "Alice", "score": 85},
    {"name": "Bob",   "score": 92},
    {"name": "Charlie", "score": 64},
    {"name": "Dave", "score": 78},
    {"name": "Ellen", "score": 88}
]
```

### 出力例
```
平均点: 81.4
最高得点者: Bob (92点)
```

### ヒント
- `json.load(f)` でJSONファイルを辞書のリストとして読み込める
- リストを `for` 文で回して各辞書の `score` を取り出す
- 平均 = 合計 ÷ 人数（`len(results)`）
- 最高得点者の探索には `max()` 関数と `key` 引数が使える: `max(results, key=lambda x: x["score"])`
